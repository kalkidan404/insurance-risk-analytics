# End-to-End Insurance Risk Analytics & Predictive Modeling

## Project Overview

This project analyzes historical automobile insurance data from South Africa to identify low-risk customer segments, optimize pricing strategies, and develop predictive risk models.

The project includes:

- Exploratory Data Analysis (EDA)
- Statistical Hypothesis Testing
- Data Version Control (DVC)
- Predictive Modeling
- Risk-Based Pricing
- Model Interpretability with SHAP

## Business Objective

AlphaCare Insurance Solutions (ACIS) aims to improve profitability and customer acquisition through data-driven insurance pricing and risk segmentation.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- XGBoost
- SHAP
- DVC
- GitHub Actions
